observedEvents

Define the events which will be emitted by the wrapped component. Bubbling events will bubble through, but non-composed events from ShadowDOM elements will not and will be re-dispatched from the proxy.

allowEvent()

Override this method to stop events from being bubbled through this element.
Source:
  • proxy-component.js, line 72
Example

filter specific events out

class MyWidget extends ProxyComponent {
  allowEvent(ev) {
    // don't propagate click events for the v2 component
    return this.getTargetElement() !== `my-widget-v2` && event.type !== `click`;
  }
}

getTargetElement() → {string}

Override to determine which tag to instantiate as the child. This is where all switching logic should go.
Source:
  • proxy-component.js, line 49
Returns:
Type
string
Example

a URL based feature flag

class MyWidget extends ProxyComponent {
  getTargetElement() {
    if (window.location.search.includes('enable_beta') {
      return 'my-widget-v2';
    }

    return 'my-widget-v1'
  }
}
